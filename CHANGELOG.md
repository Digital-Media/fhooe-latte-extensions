## [1.0.0] - 2026-03-24

### Added

- Initial release.
- Added `RouterExtension` that allows using paths from the router in Latte templates via `url_for()` and `get_base_path()`.
- Added `SessionExtension` that allows easy access to `$_SESSION` in Latte templates via `session()`.
- Added unit tests with Pest.
- Added [phpstan](https://packagist.org/packages/phpstan/phpstan) for code analysis.
- Added `README.md`.
- Added notes on Contributing.
- Added this changelog.
